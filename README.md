# Phishing-Threat-Analysis-MITRE-ATT-CK-Mapping
Investigated live phishing URLs using PhishTank and VirusTotal to simulate SOC triage workflows. Analyzed malicious infrastructure and attacker tactics (brand impersonation, typosquatting, URL obfuscation). Mapped findings to MITRE ATT&amp;CK and produced a mitigation-focused threat report aligned with enterprise security practices.
